How to embed video in HTML


STEP 1: COPY YOUR EMBED CODE

First is to locate the service hosted videos website, using YouTube as a case study, Simply search the video to be embedded and play, right click and a pop up modal appears, then click copy embed code.


STEP 2: PASTE THE EMBED CODE INTO YOUR HTML

Next,simply paste the code into the page, post, or section of your website where you’d like to embed the video.. Save or publish and you’re good to go. The video should display inline just where you placed the code.
